This folder holds the .po files which are used for localization. These files
are automatically updated by lazarus during a clean build. For each compiled
localizable unit a corresponding .lrt file is generated. This information is
used to update the project's .po file.

When Notepas is started it looks for the most appropriate .po file to use to
translate resourcestrings and component captions. The language to start the
application with is determined by the LanguageCode property in the configuration
file (settings.xml).

Files in this folder:

 - notepas.po
     The default translation file which can be used as a template to add
     a new language.

 - notepas.<languagecode>.po
     These holds the translations for the language corresponding to the
     languagecode.

As .po files are just textfiles they can be edited by any text editor. Another
option is to use Poedit (http://www.poedit.net/) which eases the translator's
job by keeping track of common translations.
